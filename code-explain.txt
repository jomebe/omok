콘솔 기반 인터페이스에서 15x15 게임 보드에서 두 플레이어가 차례로 자신의 기물을 배치합니다. 플레이어는 'X'와 'O'로 표시되며, 'X'부터 시작합니다.

게임은 'X'나 'O'가 5개 이상의 연속된 가로, 세로, 대각선으로 나란히 배치되면 끝납니다.

코드에는 다음과 같은 내용이 포함되어 있습니다.

1.
UP, DOWN, LEFT, RIGHT 키를 ASCII 값으로 정의합니다.
2.
필요한 헤더 파일을 포함합니다.
3.
gotoxy(int x, int y) 함수: 콘솔의 커서 위치를 설정합니다.
4.
dx와 dy 배열: 8방향 검사를 위한 좌표를 정의합니다.
5.
field 2차원 배열: 게임 보드를 나타냅니다.
6.
print_field() 함수: 콘솔에 게임 보드를 출력합니다.
7.
checkfive(int color) 함수: 5목(고무오미)을 검사합니다.
8.
main() 함수: 게임을 진행합니다.
게임 보드를 초기화하고, 시작 플레이어를 설정합니다.
무한 루프를 사용하여 게임을 계속합니다.
현재 플레이어에게 이동할 위치를 입력하도록 요청합니다.
이동이 유효한지 확인하고, 게임 보드를 업데이트합니다.
checkfive() 함수를 호출하여 현재 플레이어가 5목을 만들었는지 확인합니다.
플레이어가 5목을 만들었거나, 유효하지 않은 이동을 했을 경우, 해당 플레이어에게 다시 이동할 위치를 입력하도록 요청합니다.
5목을 만들거나, 유효하지 않은 이동을 반복하지 않으면, 차례를 다음 플레이어로 넘깁니다.


코드에는 오류 처리도 포함되어 있습니다. 예를 들어, 이미 채워진 셀에 기물을 놓거나, 게임 보드 경계를 벗어나는 이동을 방지합니다. 또한, 6목(5목을 만들고, 한 줄에 하나의 빈 칸이 있는 경우)을 방지합니다.
